В архиве Data.zip содержится предобработанный набор данных weather_clean.csv,
сформированный на основе HTML-страниц архива погоды аэропортов России
(формат METAR, источник rp5.ru).

В ходе работы выполнены:
- сбор данных из всех предоставленных HTML-файлов;
- формирование единой структуры данных;
- очистка данных от дубликатов и пропусков;
- приведение числовых признаков к корректным типам данных;
- добавление новых признаков для анализа метеоусловий.
