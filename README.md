# DEMO-EXAM

Модуль 1: Сбор данных, предобработка данных и
визуализация зависимостей
Исходные файлы:
1) Архив погоды.zip (Данные для обработки)
2) Модуль 1.pdf (Инструкция к 1 модулю)
Результаты работы:
1) Data.zip (Предобработанный набор данных)
2) Report_М1.html, Report_М1.ipynb (или Report_М1.py) (Отчет о
проделанной работе)
3) Readme.txt (Дополнительные комментарии)

ВВЕДЕНИЕ
Вам предоставлены исходные данные для выполнения этого модуля.
Архив содержит файлы веб-страниц с информацией об архиве погоды в
нескольких аэропортах России за 3 суток. В этом модуле Вам необходимо
сформировать датасет, содержащий данные о погодных условиях по всем из
предоставленных файлов и провести визуальный анализ данных.

ИНСТРУКЦИЯ УЧАСТНИКУ
Убедитесь, что вы сохранили отчет о проделанной работе. К концу этой
сессии у вас должны быть достигнуты следующие практические результаты:

ПРАКТИЧЕСКИЕ РЕЗУЛЬТАТЫ

1.1 Выделение итогового набора полей для одной записи
Ознакомьтесь внимательно с файлом описания данных.
Архив содержит в себе веб-страницы с информацией о погодных
условиях в 8 аэропортах России. Погодные условия сильно влияют на
деятельность аэропортов, поэтому анализ метеоданных очень важен.
Необходимо проанализировать представленные данные и определить
итоговый набор полей для создания структуры данных, в которую
преобразованные записи будут импортированы.

2

1.2 Создание структуры данных для итогового набора полей
Необходимо создать структуру данных, которая позволит сохранить
записи, состоящие из итогового набора полей. Для полученной структуры
данных дать описание её полей, отобразить в отчете.
Сформируйте набор данных о погодных условиях в соответствии с
разработанной структурой по всем предоставленным данным.

1.3 Предобработка данных
Используя программные средства, проверьте данные на наличие
дубликатов, пропусков и уберите их, если они присутствуют.

1.4 Визуальный анализ данных
Используя программные средства, визуализируйте зависимости
атрибутов в наборе данных. Также необходимо построить графики
распределения всех признаков из итогового набора полей. Приведите
интерпретацию полученным результатам.
Проведите визуальный анализ данных, отображающий
закономерности и нестандартные ситуации в наборе данных. Используйте
разные типы диаграмм.

1.5 Конструирование признаков (Feature Engineering)
Добавьте новые признаки (1-2), которые на ваш взгляд позволят
улучшить качество данных с точки зрения прогноза метеоусловий и
движения воздушных судов, обоснуйте. Напишите функцию для создания
новых признаков и примените ее к исходным данным. Проведите визуальное
исследование полученных признаков.

1.6 Подготовка отчета
Подготовьте отчет о проделанной работе по итогам модуля, в котором
будут представлены результаты, выводы и обоснования выбора по каждому
разделу задания. В отчете также опишите содержимое результирующих
файлов архива Data.zip

19.01.2024. Данные соберите в один набор данных.
Сформируйте набор данных о погодных условиях в соответствии с
разработанной структурой по всем предоставленным данным.
